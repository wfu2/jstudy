# 推送内容模板

用于生成每日推送内容的格式规范（约 1500-2000 字）。

## 结构

```markdown
📅 第X週 Day Y ｜ N? ｜ 主题

━━━━━━━━━━━━━━━━━━━━━━

🔤 今日詞彙（3+1）

単語（たんご）[词性]：中文释义
※ 日常补充分（如果是）

━━━━━━━━━━━━━━━━━━━━━━

📐 語法點：标题

核心解释（1-2句日语结构说明）
例句，全部汉字注音。

━━━━━━━━━━━━━━━━━━━━━━

🎬 動畫例句（《作品名》）

「日本語の文（全日文，汉字全部注音）」
→ "中文翻译"
→ 💡 学法提示

━━━━━━━━━━━━━━━━━━━━━━

💡 每日練習

1句练习（填空/翻译），答案附在后面（跨行）

━━━━━━━━━━━━━━━━━━━━━━

📌 明日預告：简要说明
```

## 注音规则

见 README.md「注音规则」节。

## 字数参考

- 词汇部分：3-4个日语词汇，每个约30-50字（含注音+释义）
- 语法点：1个核心语法，约300-400字
- 动画例句：1个例句 + 分析，约200-300字
- 练习：1题 + 答案，约100-150字
- 分隔符/标题：约150字
- 合计：约1500-2000字
